## Technical Support

**The primary means to ask questions and get information related to Trilinos capabilities is through the [Trilinos GitHub Repository](https://github.com/trilinos/Trilinos). Many Trilinos developers actively monitor the issues and discussions and can respond quickly to your questions.**

If you have technical questions or problems using Trilinos, please
  - Submit an [issue](https://github.com/trilinos/Trilinos/issues) on the Trilinos GitHub repository, or
  - Begin a [discussion](https://github.com/orgs/trilinos/discussions) in the Trilinos GitHub Discussions.

If you know the specific [capability](capabilities.html) or [package](packages-by-area.html) you are working with, you may want to directly contact the relevant area or package leads.
