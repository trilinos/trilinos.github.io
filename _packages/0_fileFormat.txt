This folder contains one file per package.
Each file has the format

---
name: MueLu
packageSourceDirectory: muelu
link: muelu
area: solvers
githubTeam: muelu
lead: <a href="mailto:jhu@sandia.gov">Jonathan Hu</a> (<a href="https://github.com/jhux2">@jhux2</a>) 
description: Designed to solve large sparse linear systems of equations arising from PDE discretizations, providing multigrid solvers and preconditioners.
documentation: Doxygen
---

"name" - This is how the package will appear on the webpage.
"link" - The main link for the package. Can be internal or external.
"area" - Package area
"githubTeam" - The team handle on github without @
"lead" - html giving information about the team lead(s)
"description" - A short package description
"documentation": link to the documentation or special string "Doxygen"
