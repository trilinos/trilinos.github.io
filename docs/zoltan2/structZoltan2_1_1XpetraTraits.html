Detailed Description

template<typename User>
struct Zoltan2::XpetraTraits< User >

Defines the traits required for Tpetra, Eptra and Xpetra objects.

Definitions are provided for:

  • Tpetra::CrsMatrix<scalar_t, lno_t, gno_t, node_t>
  • Epetra_CrsMatrix
  • Xpetra::CrsMatrix<scalar_t, lno_t, gno_t, node_t>
  • Xpetra::CrsMatrix<double, int, int, node_t>
  • Tpetra::CrsGraph<lno_t, gno_t, node_t>
  • Epetra_CrsGraph
  • Xpetra::CrsGraph<lno_t, gno_t, node_t>
  • Xpetra::CrsGraph<int, int, node_t>
  • Tpetra::Vector<scalar_t, lno_t, gno_t, node_t>
  • Epetra_Vector
  • Xpetra::Vector<scalar_t, lno_t, gno_t, node_t>
  • Xpetra::Vector<double, int, int, node_t>
  • Tpetra::MultiVector<scalar_t, lno_t, gno_t, node_t>
  • Epetra_MultiVector
  • Xpetra::MultiVector<scalar_t, lno_t, gno_t, node_t>
  • Xpetra::MultiVector<double, int, int, node_t>
