Detailed Description

Definition of the divergence-free polynomial basis.

Let \(NB_d=\) the number of basis components in the scalar Taylor polynomial basis, for the same dimension \(d\) and up to the same degree of polynomial. Let \(NB_{(d-1)/v}=\) the number of basis components in the scalar Taylor polynomial basis, for one dimension lower \((d-1)\) that does not use same contain the direction v.

  • in 1D) No definition.
  • in 2D) The first \(NB_d\) elements are

    \[\begin{bmatrix}b_i\\ -\int \frac{\partial}{\partial x}(b_i)\; dy\end{bmatrix},\]

    where \(i\) is the basis element number for the divergence-free polynomial basis.\newline The next \(NB_{(d-1)/y}\) elements are

    \[\begin{bmatrix}0\\b_j\end{bmatrix},\]

    where \(j\) is the basis element number for the divergence-free polynomial basis of one dimension lower using only the variable \(x\).
  • in 3D) The first \(NB_d\) elements are

    \[\begin{bmatrix}b_i\\ 0 \\-\int \frac{\partial}{\partial x}(b_i)\; dz\end{bmatrix},\]

    where \(i\) is the basis element number for the divergence-free polynomial basis.\newline The next \(NB_d\) elements are

    \[\begin{bmatrix}0\\ b_i \\-\int \frac{\partial}{\partial y}(b_i)\; dz\end{bmatrix},\]

    where \(i\) is the basis element number for the divergence-free polynomial basis.\newline The next \(NB_{(d-1)/z}\) elements are

    \[\begin{bmatrix}0\\0\\b_j\end{bmatrix},\]

    where \(j\) is the basis element number for the divergence-free polynomial basis of one dimension lower using only the variables \(x\) and \(y\).
