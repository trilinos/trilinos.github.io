Detailed Description

template<class Node = typename SmootherPrototype<double, int, int>::node_type>
class MueLu::AmesosSmoother< Node >

Class that encapsulates Amesos direct solvers.

This class creates an Amesos preconditioner factory. The factory is capable of generating direct solvers based on the type and ParameterList passed into the constructor. See the constructor for more information.

template<class Node >
MueLu::AmesosSmoother< Node >::AmesosSmoother ( const std::string &  type = "",
const Teuchos::ParameterList &  paramList = Teuchos::ParameterList() 
)

Constructor.

Creates a MueLu interface to the direct solvers in the Amesos package. The options are those specified in the Amesos user's manual.

Parameters
typesolver type
listoptions for the particular solver type

Here is how to select the more commonly used direct solvers:

  • KLU (serial sparse direct solver)
    • type = Amesos-KLU
    • parameter list options
      • none required
  • SuperLU (serial sparse super-nodal direct solver)
    • type = Amesos-SuperLU
    • parameter list options
      • none required

If you are using type=="", then either SuperLU or KLU are used by default.

See also Amesos_Klu and Amesos_Superlu.
