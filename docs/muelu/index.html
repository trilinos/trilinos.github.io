MueLu Overview

MueLu is a flexible and extensible multigrid solver framework. It is intended for the research and development of new multigrid preconditioning techniques. More information can be found at https://trilinos.org/packages/muelu.

Documentation

Useful views of the Doxygen documentation are:

  • Browsing the entire class documentation. Choose the Class Index subtab.
  • Browsing the class documentation organized by logical groups. Choose the Modules tab.

The MueLu User's Guide is located in muelu/doc/UsersGuide and at the MueLu home page.
