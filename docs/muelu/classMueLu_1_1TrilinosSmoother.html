Class that encapsulates external library smoothers.

Autoselection of Ifpack or Ifpack2 according to the underlying linear algebra library.

Parameters
[in]typeSmoother type. Can currently be
  • RELAXATION for (symmetric) Gauss-Seidel or Jacobi smoothing
  • CHEBYSHEV for Chebyshev polynomial smoothing
  • ILU for incomplete LU smoothing
[in]paramListA list holding parameters understood by either Ifpack or Ifpack2 to guide the smoother setup. See MueLu::Ifpack2Smoother for summary of the most commonly used parameters. See the Ifpack/Ifpack2 documentation for the full set.
