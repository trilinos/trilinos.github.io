const FactoryManagerMap &  factoryManagersIn 
) const
virtual

: Interpret Factory parameter list and build new factory

Parameters
param[in]: ParameterEntry being either the parameter list containing the "factory" parameter declaring the factory type (e.g., "TrilinosSmoother") or being a plain Parameter containing the factory type as value
factoryMapIn[in]: FactoryMap containing a map between factory name (e.g., "smootherFact1") and corresponding factory of all previously defined factories
factoryManagersIn[in]: FactoryManagerMap containing a map between group names and Factory manager objects. Needed for factories with sub-factory managers.

Parameter List Parsing:

<Parameter name="smootherFact0" type="string" value="TrilinosSmoother">

or: 

<ParameterList name="smootherFact1">
  <Parameter name="factory" type="string" value="TrilinosSmoother"/>
  ...
</ParameterList>
