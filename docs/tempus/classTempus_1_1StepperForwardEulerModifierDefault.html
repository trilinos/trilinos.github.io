Detailed Description

template<class Scalar>
class Tempus::StepperForwardEulerModifierDefault< Scalar >

Default modifier for StepperForwardEuler.

The default modifier provides no-op functionality for the modifier. See StepperForwardEulerModifierBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
