Detailed Description

template<class Scalar>
class Tempus::StepperNewmarkImplicitDFormModifierDefault< Scalar >

Default modifier for StepperNewmarkImplicitDForm.

The default modifier provides no-op functionality for the modifier. See StepperNewmarkImplicitDFormModifierBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
