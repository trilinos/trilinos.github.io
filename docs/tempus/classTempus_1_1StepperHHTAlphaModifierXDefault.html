Detailed Description

template<class Scalar>
class Tempus::StepperHHTAlphaModifierXDefault< Scalar >

Default ModifierX for StepperHHTAlpha.

The default provides no-op functionality for ModifierX. See StepperHHTAlphaModifierXBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
