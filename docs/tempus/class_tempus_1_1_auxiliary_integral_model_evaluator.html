Detailed Description

template<typename Scalar>
class Tempus::AuxiliaryIntegralModelEvaluator< Scalar >

ModelEvaluator for integrating auxiliary equations.

Given a ModelEvaluator defining an auxiliary/response function g(x_dot,x,p,t) this ModelEvaluator defines a new ODE dz/dt = g(x_dot(t),x(t),p,t) used to compute the integral of g over some time range. This ModelEvaluator can then be used by Tempus to compute the definite integral.
