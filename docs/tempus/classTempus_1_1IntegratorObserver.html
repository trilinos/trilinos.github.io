Detailed Description

template<class Scalar>
class Tempus::IntegratorObserver< Scalar >

IntegratorObserver class for time integrators.

This is a means for application developers to perform tasks during the time integrator, e.g.,

  • Compute specific quantities
  • Output information
  • Adjust the time step (CFL stability)
  • "Massage" the working solution state
  • ...

Design Considerations

  • IntegratorObserver should have access to the entire SolutionHistory, as application developers may have that need.
  • The needed IntegratorObserver functions are determined by the access needs in Integrator::advanceTime().
  • IntegratorObserver is not stateless! Developers may touch the solution state! Developers need to be careful not to break the restart (checkpoint) capability.
