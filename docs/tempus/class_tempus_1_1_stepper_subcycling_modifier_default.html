Detailed Description

template<class Scalar>
class Tempus::StepperSubcyclingModifierDefault< Scalar >

Default modifier for StepperSubcycling.

The default modifier provides no-op functionality for the modifier. See StepperSubcyclingModifierBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
