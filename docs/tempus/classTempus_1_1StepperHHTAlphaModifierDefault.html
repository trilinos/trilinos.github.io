Detailed Description

template<class Scalar>
class Tempus::StepperHHTAlphaModifierDefault< Scalar >

Default modifier for StepperHHTAlpha.

The default modifier provides no-op functionality for the modifier. See StepperHHTAlphaModifierBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
