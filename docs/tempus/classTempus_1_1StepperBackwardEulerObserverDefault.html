Detailed Description

template<class Scalar>
class Tempus::StepperBackwardEulerObserverDefault< Scalar >

Default observer for StepperBackwardEuler.

The default observer provides no-op functionality for the observer. See StepperBackwardEulerObserverBase for details on the algorithm.

Applications can copy this implementation, rename, implement their action, and set on the stepper to get app-specific functionality.
