Detailed Description

Adapter class for creating a symmetric operator from an Epetra_MultiVector.

This class will apply the operation [default] or , for the Apply method of the Epetra_Operator / Anasazi::Operator. The Anasazi::EpetraSymMvOp operator is useful when trying to compute a few singular values of the Epetra_MultiVector . The singular values are the square-root of the eigenvalues of and .

Note
The Epetra package performs double-precision arithmetic, so the use of Epetra with Anasazi will only provide a double-precision eigensolver.
