Detailed Description

Adapter class for creating a weighted symmetric operator from an Epetra_MultiVector and Epetra_Operator.

This class will apply the operation for the Apply method of the Anasazi::Operator. The Anasazi::EpetraW2SymMvOp operator is useful when trying to compute a few singular values of the Epetra_MultiVector under the weighting matrix .
The singular values are the square-root of the eigenvalues of .

Note
The Epetra package performs double-precision arithmetic, so the use of Epetra with Anasazi will only provide a double-precision eigensolver.
