Detailed Description

WorkspaceStore class that can be used to actually reinitialize memory.

The client can create concrete instances of this type and initialize the memory used. The client should call initialize(num_bytes) to set the number of bytes to allocate where num_bytes should be large enough to satisfy all but the largests of memory request needs.
