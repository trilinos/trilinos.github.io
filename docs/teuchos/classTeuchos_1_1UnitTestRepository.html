Detailed Description

Singleton unit testing repository.

This class is the universal driver for unit testing. This class should be almost completely invisible to a user of the test harness. The main interaction is through command-line arguments set and processed by calling runUnitTestsFromMain() from a main() function. See –help for details of the options. For a more general overview, see Unit Testing Support.

bool Teuchos::UnitTestRepository::verboseUnitTests ( )

Returns if unit tests are verbose or not.

This can be used in individual unit tests that need to know if the unit test harness is running in verbose mode. This is useful when the unit test's std::ostream 'out' can not be printed to (for example, when Fortran is testing is running).
