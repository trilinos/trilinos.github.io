Detailed Description

template<class T>
class Teuchos::EnhancedNumberTraits< T >

Class defining the traits of the number type being used in an EnhancedNumberValidator.

This class defines some of the traits of a number type being used by an EnhancedNumberValidator. The number has the following traits:

  • min Defines the minimum possible value the number type can take on.
  • max Defines the maximum possible value the number type can take on.
  • defaultStep Defines the default amount a value of the number type should be incremented by when being incremented in a UI.
  • defaultPrecision Defines the default number of decimals with which the number type should be displayed in a UI. This trait is useless for non-floating point number types.

Note that simply using this class will result in compile time errors. Only specializations of this class will produce valid code.
