Detailed Description

template<typename EvalT, typename TRAITS>
class panzer::MultiVariateParameter< EvalT, TRAITS >

Constant parameter from sacado parameter library.

Represents a sum of n independent parameters. Useful for SA/UQ scaling studies where the number of parameters must be varied.
