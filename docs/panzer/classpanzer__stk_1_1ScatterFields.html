Detailed Description

template<typename EvalT, typename TraitsT>
class panzer_stk::ScatterFields< EvalT, TraitsT >

This class is a scatter operation to the mesh. It takes a set of field names and basis objects and then writes them to the mesh object. Note that scaling vector must be the same length as the names vector. The scaling is applied to each field.
