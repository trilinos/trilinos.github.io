Detailed Description

template<typename TRAITS>
class panzer::DOF_PointValues< typename TRAITS::Jacobian, TRAITS >

Interpolates basis DOF values to IP DOF Curl values (specialization for the jacobian) Allows short cut for simple jacobian to dof structure.
