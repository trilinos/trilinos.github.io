Tag only Constructor.

Perform a linear combination of fields using only input tags and double vectors.

Parameters
[in]inputsTags associated with rank 2 arrays to be summed
[in]scalar_valuesVector of length inputs.size() or zero. If the length is equal to inputs.size() each entry is scaled. Otherwise 1.0 is assumed.
[in]outputDestination array tag for the summation.
