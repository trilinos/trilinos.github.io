Detailed Description

template<class Scalar, class ArrayScalar>
class Intrepid::Basis_HGRAD_TRI_Cn_FEM_ORTH< Scalar, ArrayScalar >

Implementation of the default H(grad)-compatible orthogonal basis (Dubiner) of arbitrary degree on triangle.

Remarks
  • All degrees of freedom are considered to be internal (ie not assembled)
