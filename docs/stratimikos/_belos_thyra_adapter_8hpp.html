Detailed Description

Thyra specializations of MultiVecTraits and OperatorTraits.

This file implements partial specializations of Belos' traits classes for Thyra's abstract linear algebra interface classes. In particular, the file specializes Belos::MultiVecTraits for Thyra::MultiVectorBase, and Belos::OperatorTraits for Thyra::LinearOpBase.
