Detailed Description

template<class Real>
class ROL::LinearObjective_SimOpt< Real >

Provides the interface to evaluate linear objective functions.

This class implements the linear objective function

\[ f(x) = \langle c, x\rangle_{\mathcal{X}^*,\mathcal{X}} \]

for fixed \(c\in\mathcal{X}^*\).
