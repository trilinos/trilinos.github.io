Detailed Description

template<typename Real>
class ROL::SerialObjective< Real >

Evaluates ROL::DynamicObjective over a sequential set of time intervals.

\[ f(u,z) = \sum\limits_{k=1}^n f_k(u_{k-1},u_k,z_k) \]

\[ \frac{\partial f}{\partial u_j} = \frac{\partial f_j(u_{j-1},u_j,z_j}{\partial u_j} + \frac{\partial f_{j+1}(u_j,u_{j+1},z_{j+1}}{\partial u_j} \]
