class Ifpack2::BorderedOperator< Scalar, LocalOrdinal, GlobalOrdinal, Node >

Ifpack2 bordered operator.

Ifpack2::BorderedOperator is a concrete class extending Tpetra::Operator and defining an interface.

This class extends Tpetra::Operator, providing the additional methods:

  • compute() computes everything required to apply the bordered operator, using matrix values (and assuming that the sparsity of the matrix has not been changed);
  • isComputed() should return true if the bordered operator has been successfully computed, false otherwise.
  • getRHS() returns a reference to the matrix to be preconditioned.
  • getLHS() returns a reference to the matrix to be preconditioned.

The bordered operator is applied by apply() (which returns if isComputed() is false). Each time compute() is called, the object re-computes the actual values of the bordered operator.
