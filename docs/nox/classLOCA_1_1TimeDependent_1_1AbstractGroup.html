Detailed Description

Interface to underlying groups for time dependent systems.

This abstract class provides an interface for time dependent problems, i.e., problems with a mass matrix (typically used in eignvalue or Hopf calculations). It provides pure virtual methods for computing and manipulating the shifted matrix where is the Jacobian matrix and is the mass matrix.
