Detailed Description

An Epetra operator for solving extended sets of equations using Householder transformations.

This class implements the operator as described in the LOCA::BorderedSolver::EpetraHouseholder documentation for solving an extended set of equations. It uses the factor from a QR factorization using the compact WY representation.
