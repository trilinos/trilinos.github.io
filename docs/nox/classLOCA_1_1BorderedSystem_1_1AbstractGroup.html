Detailed Description

An interface for groups that are bordered systems.

This class provides an interface for groups whose Jacobian is of the form

where and are multivectors and is a dense matrix. It provides methods for determining the width of the bordered rows/columns and for extracting these components. It is intended to be a recusive interface, in that if the group representing also has this form, the extracted rows/columns should be the combined rows/columns of this group and the underlying group (and so on).
