Detailed Description

Tangent predictor strategy

This class implements a predictor strategy based on computing the tangent to the continuation curve. If is the vector of continuation parameters, then the solution component of the tangent vectors are computed by solving

The parameter component is set to the identity matrix.

Evaluate predictor with step size stepSize.

This method computes result[i] = xVec[i] + stepSize[i] * v[i] for each i, where v[i] is the ith predictor direction.
