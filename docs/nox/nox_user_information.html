NOX User Information

Overview

To get started with NOX, you'll want to follow these four steps.

  • Step 1: Download, Configure, Compile, and Install NOX
  • Step 2: Create concrete implementations of the NOX::Abstract classes
  • Step 3: Call NOX from your code
  • Step 4: Link your code to NOX

Generating Documentation

The documentation is available online at http://trilinos.sandia.gov/packages/nox/.

To generate a local copy of the documentation, you'll need to have doxygen installed. Then follow the instructions in nox/doc/README.

The documentation is generated in the source directory tree. The main page is nox/doc/html/index.html.
