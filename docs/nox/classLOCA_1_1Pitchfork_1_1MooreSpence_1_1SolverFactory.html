Detailed Description

Factory for creating solver objects for solving Moore-Spence pitchfork equations.

The parameters passed to the create() through the solverParams argument method should specify the "Solver Method" as described below, as well as any additional parameters for the particular strategy.

  • "Solver Method" - Name of the method. Valid choices are
    • "Salinger Bordering" (LOCA::Pitchfork::MooreSpence::SalingerBordering) [Default]
    • "Phipps Bordering" (LOCA::Pitchfork::MooreSpence::PhippsBordering)
