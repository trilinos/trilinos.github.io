Detailed Description

Failure test based on the maximum number of continuation steps.

Let denote the current number of iterations (accessed via LOCA::Stepper::getNumTotalSteps) and denote the tolerance set in the constructor of this status test. This test returns LOCA::StatusTest::Failed if Otherwise, it returns LOCA::StatusTest::NotFinished.

Note
If checkStatus is called with the type set to LOCA::StatusTest::None, it then the status is set to to LOCA::Status::Unevaluated and returned. (Also niters is set to -1.)
